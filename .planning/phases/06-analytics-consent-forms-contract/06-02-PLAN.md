# Plan 02 — `analytics.js`, normalized dataLayer pushes, check script

## Objective

Meet **ANLY-02**, **ANLY-03**, **ANLY-04**: add **`js/analytics.js`** with **`TMAnalytics.track(name, params)`** (exact global per implementation) that merges **`analytics-labels.json`** constants, **`event_id`** (prefer `crypto.randomUUID` with fallback), ISO **timestamp**, **`page_path`**, and **consent snapshot**; forbid PII keys; add **`scripts/check-analytics-contract.js`** and **`npm run check:analytics`**.

## Preconditions

- Plan 01 merged — `dataLayer` and GTM bootstrap exist.
- `src/data/site/analytics-labels.json` is source of allowed **event names / param keys** (import at build **or** duplicate minimal constants in JS — choose one; PLAN prefers **single JSON** consumed by check script).

**Files:**
