| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-01-01 | Tampering | `<a href="#main">` skip-link | accept | Anchor target is same-page hash fragment; no JS click handler attached; cannot redirect cross-origin. SiteLayout.astro injection is server-rendered, not user-controllable. |
| T-10-01-02 | Information Disclosure | `<main id="main" tabindex="-1">` | accept | Element ID is well-known (WCAG pattern); no sensitive data in attribute values. |
| T-10-01-03 | Tampering | aria-hidden sweep on SVGs | accept | Attributes are static HTML emitted at build time; no runtime computation; cannot be tampered with by user input. |

No HIGH-severity threats. All changes are server-side static HTML markup with no new dynamic surface, no eval, no user input, no third-party scripts.
